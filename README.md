# 📊 Financial Performance Report — Power BI

> An interactive dashboard analysing sales, profit, and market performance
> across products, segments, and countries for **2013–2014**.

---

## 📁 Repository Structure

```
financial-report-powerbi/
├── Financial_Report.pbix       # Power BI report
├── Financial_Sample.xlsx       # Source dataset (700 records)
└── README.md
```

---

## 🗂️ About the Report

This Power BI report was built using the **New Executive** theme and provides a single-page interactive dashboard with the following visuals:

- **Line Chart** — Profit trend over time
- **Azure Map** — Profit distribution by country
- **Clustered Column Chart** — Sales by product and segment
- **Slicer** — Interactive filter for dynamic exploration

---

## 📂 Dataset

**Source file:** `Financial_Sample.xlsx`

| Field | Details |
|---|---|
| Records | 700 transactions |
| Period | September 2013 – December 2014 |
| Countries | Canada, France, Germany, Mexico, United States of America |
| Products | Amarilla, Carretera, Montana, Paseo, VTT, Velo |
| Segments | Government, Enterprise, Midmarket, Small Business, Channel Partners |
| Total Sales | $118,726,350 |
| Total Profit | $16,893,702 |

## 🚀 Getting Started

1. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
2. Clone this repository
   ```bash
   git clone https://github.com/Hosen-GIT/financial-report-powerbi.git
   ```
3. Open `Financial_Report.pbix` in Power BI Desktop
4. If prompted, update the data source path:
   **Home → Transform Data → Data Source Settings** → point to `Financial_Sample.xlsx`
5. Click **Refresh**
